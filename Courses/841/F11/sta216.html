STA 216 - Generalized linear models (Fall 2011)

Course description

Generalized linear models (GLMs) are a powerful set of tools for analyzing categorical data. In this class we will investigate the general concepts behind these models, and survey a variety of different models in the particular contexts for which they are suited. Some examples include models for binary data, polytomous data (ordered and unordered), count data, contingency tables, and matrix data. These models cover a wide range of applications, from classical to modern. We will cover models involving random/mixed effects and (if time permits) some recent methodological development for coping with the increasing dimensionality and complexity in modern data sets. These include regularized/penalized methods and nonparametric/semiparametric models. The primary goal is to learn how to apply GLM and its modern extensions in real data analysis while acquiring an understanding of the underlying concepts.

Prerequisites
